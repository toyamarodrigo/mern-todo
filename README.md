# Simple TODO using stack MERN

## Folder structure
```
src
  |_clients             // Front
  |_models              // Model todo schema
      |_task.js
  |_public              // Static files
      |_index.html
  |_routes              // Define operations from URLs (API)
      |_task.routes.js  
  |_database.js         // Connection to Database
  |_index.js            // Server
```
